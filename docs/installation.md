# Installation

Base package:

```bash
pip install django-grpc-extra
```

Optional extras:

```bash
pip install "django-grpc-extra[codegen]"
pip install "django-grpc-extra[health]"
pip install "django-grpc-extra[reflection]"
pip install "django-grpc-extra[reload]"
```

Development install:

```bash
pip install -e ".[dev,codegen,health,reflection,reload]"
```
